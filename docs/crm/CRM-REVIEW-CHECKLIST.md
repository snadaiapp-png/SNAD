# CRM Review Checklist

- Global product scope accepted.
- CRM exclusions and source-of-truth ownership accepted.
- Domain and package boundaries accepted.
- Multi-tenant controls accepted.
- Privacy, consent, retention, and residency impacts reviewed.
- API and event contracts accepted.
- Workflow and AI integration reviewed.
- MVP backlog and sprint order accepted.
- Test and quality plan accepted.
- Product, engineering, security, data, QA, and operations owners assigned.
- Platform Core evidence linked.
- Explicit CRM build decision recorded.
