## 1. Definitions

| Metric            | Definition                                              |
|-------------------|---------------------------------------------------------|
| MRR               | Monthly Recurring Revenue from subscriptions            |
| ARR               | MRR × 12                                                |
| ARPU              | MRR / paying users                                      |
| CAC               | Sales + marketing spend / new customers acquired        |
| LTV               | ARPU × gross margin × (1 / churn rate)                  |
| Churn             | Lost MRR / prior period MRR                             |
| Retention          | 1 − churn                                               |
| Expansion Revenue | Upsell + cross-sell MRR in period                       |
| Activation Rate   | Activated tenants / onboarded tenants                   |
| Trial Conversion  | Paid tenants / trial tenants                            |
| Time to Value     | Hours from onboarding to first value-producing action   |
| Support SLA       | % of tickets resolved within tier SLA                   |
| Marketplace GMV   | Gross merchandise value through marketplace             |
| Partner-Sourced   | Revenue sourced by partners                             |
| AI Adoption       | Tenants with ≥ 1 active agent / total tenants           |
| Industry Adoption | Tenants with ≥ 1 installed pack / total tenants         |

---

## 2. Reporting Cadence

* Daily: MRR, new signups, trial conversions.
* Weekly: Activation, churn, support SLA.
* Monthly: Full metrics dashboard, cohort analysis.
* Quarterly: LTV/CAC, expansion, partner-sourced.
